# Quick Start Guide - Petty Cash App

## Overview

Dokumen ini merupakan panduan cepat untuk memulai menggunakan Petty Cash App yang bertujuan untuk membantu pengguna baru dalam memahami fitur-fitur dasar aplikasi, yaitu: mulai dari registrasi, setup saldo awal, hingga pencatatan pengeluaran pertama.

## Langkah 1: Registrasi Akun

### 1.1 Buka Halaman Registrasi

Akses aplikasi di browser Anda dan klik tombol **Register** atau **Daftar** di halaman utama.

### 1.2 Isi Form Registrasi

Lengkapi informasi berikut:

| Field | Keterangan | Contoh |
|-------|------------|--------|
| Name | Nama lengkap Anda | Budi Santoso |
| Email | Email aktif untuk login | budi@email.com |
| Password | Minimal 8 karakter | ******** |
| Confirm Password | Ulangi password | ******** |

### 1.3 Submit Registrasi

Klik tombol **Register** dan Anda akan diarahkan ke Dashboard.

## Langkah 2: Setup Saldo Awal

### 2.1 Perhatikan Banner Setup

Setelah login pertama kali, Anda akan melihat banner kuning yang mengingatkan untuk setup saldo awal kas kecil.

### 2.2 Klik "Set Saldo Awal"

Klik tombol **Set Saldo Awal** pada banner atau melalui menu sidebar **Tambah Dana**.

### 2.3 Isi Form Dana Awal

| Field | Keterangan | Contoh |
|-------|------------|--------|
| Jumlah | Nominal dana awal kas kecil | 1.000.000 |
| Catatan | Keterangan (opsional) | Dana awal kas kecil bulan November |
| Tanggal | Tanggal penetapan | 27/11/2025 |

### 2.4 Simpan

Klik **Set Saldo Awal** untuk menyimpan. Anda akan kembali ke Dashboard dengan saldo yang sudah terisi.

## Langkah 3: Catat Pengeluaran Pertama

### 3.1 Akses Form Pencatatan

Dari Dashboard, klik kartu **Catat Pengeluaran** atau gunakan menu sidebar.

### 3.2 Pilih Kategori

Pilih kategori yang sesuai dengan jenis pengeluaran:

| Kategori | Contoh Pengeluaran |
|----------|-------------------|
| Office Supplies | Alat tulis, kertas, tinta printer |
| Food & Beverages | Konsumsi rapat, snack kantor |
| Transportation | Parkir, bensin, ojek online |
| Miscellaneous | Pengeluaran lain-lain |
| Other | Pengeluaran yang tidak masuk kategori |

### 3.3 Isi Detail Pengeluaran

| Field | Keterangan | Contoh |
|-------|------------|--------|
| Jumlah | Nominal pengeluaran | 50.000 |
| Kategori | Pilih dari daftar | Office Supplies |
| Deskripsi | Keterangan singkat (max 200 karakter) | Beli pulpen dan kertas HVS |
| Tanggal | Tanggal transaksi | 27/11/2025 |

### 3.4 Simpan Pengeluaran

Klik **Simpan Pengeluaran**. Saldo akan otomatis terhitung ulang.

## Langkah 4: Lihat Daftar Transaksi

### 4.1 Akses Daftar Transaksi

Klik menu **Transaksi** di sidebar untuk melihat semua transaksi.

### 4.2 Filter Transaksi

Gunakan filter untuk mempersempit pencarian:

- **Tanggal Mulai**: Filter dari tanggal tertentu
- **Tanggal Akhir**: Filter sampai tanggal tertentu
- **Kategori**: Filter berdasarkan kategori

### 4.3 Edit atau Hapus Transaksi

- **Desktop**: Gunakan tombol Edit/Hapus di setiap baris transaksi
- **Mobile**: Swipe ke kiri pada item transaksi untuk reveal action buttons

## Langkah 5: Tambah Dana (Replenishment)

### 5.1 Kapan Harus Tambah Dana?

Ketika saldo kas kecil mulai menipis, Anda perlu melakukan pengisian ulang.

### 5.2 Proses Penambahan

1. Klik menu **Tambah Dana** di sidebar
2. Isi jumlah dana yang ditambahkan
3. Tambahkan catatan (opsional)
4. Pilih tanggal penambahan
5. Klik **Tambah Dana**

### 5.3 Verifikasi

Saldo di Dashboard akan otomatis bertambah sesuai jumlah yang dimasukkan.

## Tips Penggunaan

### Best Practices

1. **Catat Segera** - Catat pengeluaran segera setelah terjadi agar tidak lupa
2. **Deskripsi Jelas** - Gunakan deskripsi yang spesifik untuk memudahkan tracking
3. **Kategori Tepat** - Pilih kategori yang sesuai untuk laporan yang akurat
4. **Rekonsiliasi Rutin** - Cocokkan saldo sistem dengan uang fisik secara berkala

### Shortcuts

| Action | Desktop | Mobile |
|--------|---------|--------|
| Catat Pengeluaran | Klik kartu di Dashboard | Klik kartu di Dashboard |
| Lihat Transaksi | Menu Transaksi | Menu Transaksi |
| Edit Transaksi | Tombol Edit | Swipe kiri + tap ikon |
| Hapus Transaksi | Tombol Hapus | Swipe kiri + tap ikon |

## FAQ Quick Start

### Q: Bagaimana jika saya salah input transaksi?

A: Anda dapat mengedit atau menghapus transaksi dari halaman Daftar Transaksi.

### Q: Apakah saya bisa membatalkan penambahan dana?

A: Saat ini fitur ini belum tersedia. Pastikan input dengan benar.

### Q: Bagaimana jika saldo sudah habis?

A: Lakukan penambahan dana melalui menu **Tambah Dana**.

### Q: Bisakah saya menggunakan tanggal mundur untuk transaksi?

A: Ya, Anda dapat memilih tanggal kapan pun selama tidak di masa depan.

### Q: Apakah data saya aman?

A: Ya, data disimpan dengan aman dan hanya dapat diakses oleh akun Anda.
